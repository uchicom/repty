yaml template to pdf

snakeyamlとpdfboxを使って、PDFを出力する  
フォント指定  
線指定  
画像描画  
四角形描画（塗りつぶしも）  
線描画  
${変数名}のマッピングに対応予定  
  
可変の表も対応する。最大値やオフセット（ページまたがりは検討中）  
listパラメータとメンバ変数名でアクセスgetterメソッドが必要。
